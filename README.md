# volumeResetter-Android
メディア再生音量を0にするアプリです。

## 制作経緯
夜寝る前、スマホで動画や音楽を楽しんだ後、ボリュームを下げ忘れがちです。

翌日、学校や会社でボリューム全開に音楽などが流れた場合には、めちゃくちゃ恥ずかしい思いをします。

そんなわけで、日次または毎時バックグラウンドでボリューム０にするアプリを作ろうと思いました。

## 要件定義
動作タイミング：日次0時
アプリ起動有無：起動していなくても動作させる

設定画面を設け、好きな時間帯にリセット出来るようにする。
